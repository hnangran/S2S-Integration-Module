# S2S Integration UI Package

This package contains all user interface components for the S2S Integration Module.

## Package Contents

### Lightning Web Components (LWC)
- `connectionManager` - Manage Salesforce org connections
- `mappingConfigurator` - Configure object and field mappings
- `syncMonitor` - Monitor sync operations and status
- `syncStatusDashboard` - Dashboard for sync status visualization

### Aura Components
Legacy Aura components (if needed for compatibility)

### Lightning Pages
- Integration Manager app page
- Sync Monitor app page

### Static Resources
- CSS stylesheets
- Images and icons
- JavaScript libraries

## Dependencies
- **S2S Integration Core** (required)

## Installation Order
Install this package after the Core package.

## Package Name
S2S Integration UI
