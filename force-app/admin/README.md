# S2S Integration Admin Package

This package contains administrative components including permission sets, settings, and setup components.

## Package Contents

### Permission Sets
- `S2S_Integration_Administrator` - Full administrative access
- `S2S_Integration_User` - Standard user access for sync operations
- `S2S_Integration_Viewer` - Read-only access for monitoring

### Custom Settings
- `S2S_System_Configuration` - System-wide configuration settings

### Post-Installation Scripts
- `S2SPostInstallScript` - Automated setup after package installation

## Dependencies
- **S2S Integration Core** (required)
- **S2S Integration UI** (required)

## Installation Order
Install this package last, after Core and UI packages.

## Package Name
S2S Integration Admin
